# GoodByeMAX

Это расширение, которое блокирует доступ к официальному сайту мессенджера **MAX** и предотвращает его открытие в браузере. Расширение предназначено для пользователей, которые хотят ограничить использование данного мессенджера или минимизировать отвлекающие факторы.
